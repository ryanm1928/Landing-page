Apa itu website,bagaimanakah website dibuat?

Situs web atau website adalah kumpulan informasi dalam bentuk halaman web yang saling tergabung di sebuah domain atau URL. Contohnya, dalam sebuah website dengan domain cikenblekpaper.com, terdapat berbagai halaman lain, mulai dari blog, menu, halaman kontak, dan sebagainya yang bisa diakses langsung dari halaman beranda (homepage). Kumpulan halaman ini bisa diakses tentunya dengan koneksi internet.Pada umumnya website ada 2 macam yaitu website statis dan website dinamis website statis adalah website yang konten nya tidak berubah(tetap) seperti blog,media,organisasi,dll.
sedangkan website dinamis merupakan website yang konten nya berubah-ubah seperti E-commerce,facebook,youtube,dll lalu bagiamana website dibuat website dibuat menggunakan code yang sudah di tetapkan seperti HTML,CSS,JavaScript untuk webiste statis,untuk membuat website dinamis di tambahkan bahasa pemograman PHP,Nodejs,golang dll. namun sekarang dalam membuat webiste sudah di permudah dengan CMS
ini merupakan alat untuk membantu mengembangkan,memodifikasi,dan membuat website dengan mudah tanpa mengerti sekalipun dengan coding macam macam CMS seperti WordPress,Blogger,dll

Apakah sudah selesai sampai situ saja untuk membuat website?

Tidak,bila anda membuat web statis mungkin hanya sedikit saja yang perlu di perhatikan,tetapi jika anda membuat web dinamis ada banyak yang harus di perhatikan seperti client server,database,keamanan,bug,dll.
karna dalam web dinamis memungkinkan penggunanya untuk berinteraksi secara langsung, dalam artian pengguna dapat menambah, memodifikasi, ataupun menghapus konten di dalam sebuah web tanpa harus membuka struktur kode dari web tersebut. Karena fungsi inilah, ngoding web dinamis menjadi lebih rumit daripada ngoding web statis. Sebuah web dinamis memungkinkan untuk menampilkan sebuah halaman web yang sama tapi dengan tampilan yang berbeda kepada pengguna yang berbeda pula Web dinamis juga bisa dibilang lebih powerful karena mempunyai fitur seperti login, daftar pengguna, proses pembayaran, dan lain sebagainya. Tapi karena kaya fitur itulah proses ngoding web dinamis tidak semudah membuat web statis. Diperlukan pemahaman bahasa pemrograman yang lebih, juga memerlukan biaya yang lebih unutk maintenance, dan penambahan fitur,dll Dalam hal keamanan pun, web dinamis bisa dikatakan cukup rentan terhadap serangan cyber. Fitur seperti pembayaran dan pendaftaran data diri seseorang yang bisa saja diretas sehingga data-data tersebut bocor.
